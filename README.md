# PokeFlip

포켓몬 카드를 이용해 짝맞추기 게임을 즐기고 포켓몬 도감을 볼 수 있는 웹사이트 입니다.

<br/>

## 🔗 배포 링크

> https://pokeflip-65wt85h5g-12123s-projects.vercel.app/

<br/>


## 🗓️ 개발 기간

> 2025.03.13 ~ 2025.03.30

<br/>

## 목표

PokeApi를 통해 비동기적으로 데이터를 관리해 그 데이터를 이용해 포켓몬 카드를 만들고 짝맞추기 게임, 도감 서비스를 이용자에게 제공합니다
<br/>
